# JAVA
#### 프로그래밍 언어의 종류

+ 기계어 - 0,1로 구성된 언어 / 컴퓨터의 CPU는 기계어만 이해, 처리

+  어셈블리어 - 기계어를 표현하기 쉬운 상징적인 단어로 일대일 대응 시킨 언어

+ 고급언어 - 사람이 이해하기 쉽고 자료구조, 알고리즘을 표현하기 위해 고안된 언어
